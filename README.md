## how to use
input `todo` to startup. Press `h` for help.

- a - add new todo
- e - edit current todo
- d - delete current todo
- x - toggle completion status
- t - toggle days mode
- s - toggle sorting mode
- :w - save todos
- :q - quit and autosave
